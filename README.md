# Scroll-floor
前端滚动页面视图，左侧悬浮楼层选中状态变更+点击楼层滚动到对应视图


## 技术支持
- 前端：`Node`、`less`、`javasCript`、`jQuery`
- 服务器端：`Node`


## 运行步骤
1. `npm install`  ——下载依赖
2. ` node index` ——启动node服务
  以下提示表示成功：
  server start successful on : 3000
  浏览器中访问：
  localhost:3000/view/index.html
